◆ SO_SSL_BYPASS_X509_VERIF [1/12]

#define SO_SSL_BYPASS_X509_VERIF   0x01

Allow an opened SSL socket to bypass the X509 certificate verification process. It is highly required NOT to use this socket option in production software applications. It is supported for debugging and testing purposes. The option value should be cast to int32_t type and it is handled as a boolean flag.
