int8_t accept (SOCKET sock, struct sockaddr *addr, uint8_t *addrlen)
 

Detailed Description

The function has no current implementation. An empty declaration is used to prevent errors when legacy application code is used. For recent application use, the accept function can be safer as it has no effect and could be safely removed from any application using it.

Returns
The function returns ZERO for successful operations and a negative value otherwise. The possible error values are:
  • SOCK_ERR_NO_ERROR Indicating that the operation was successful.
  • SOCK_ERR_INVALID_ARG Indicating passing invalid arguments such as negative socket ID.
