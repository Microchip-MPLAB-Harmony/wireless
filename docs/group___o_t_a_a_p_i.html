Detailed Description

The WINC supports OTA (Over-The-Air) updates. Using the APIs described in this module, it is possible to request an ATWINC15x0 to update its firmware, or safely rollback to the previous firmware version.
There are also APIs to download files and store them in the WINC's Flash (supported by ATWINC1510 only), which can be used for Host MCU OTA updates or accessing information stored remotely.
